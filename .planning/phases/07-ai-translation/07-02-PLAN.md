AI Config Controller:
```typescript
GET  /api/ai/config/status?projectId=xxx
  → { configured: boolean, provider?: string, level?: 'project' | 'team', keyHint?: string }

GET  /api/ai/config/team/:teamId
  → { provider: string, keyHint: string, model?: string, baseUrl?: string } | null

PUT  /api/ai/config/team/:teamId
  Body: { provider: string, apiKey: string, model?: string, baseUrl?: string }
  → { success: true }

DELETE /api/ai/config/team/:teamId
  → { success: true }

GET  /api/ai/config/project/:projectId
  → { provider: string, keyHint: string, model?: string, baseUrl?: string } | null

PUT  /api/ai/config/project/:projectId
  Body: { provider: string, apiKey: string, model?: string, baseUrl?: string }
  → { success: true }

DELETE /api/ai/config/project/:projectId
  → { success: true }
```

AI Controller (updated):
```typescript
POST /api/ai/translate
  Body: { text: string, from: string, to: string[], projectId: string }
  → Record<string, string>  (or 503 if no provider configured)

POST /api/ai/generate/key
  Body: { remark: string, tag?: string, module?: string, projectId: string }
  → { success: true, data: string }  (or 503 if no provider configured)
```

Supported providers: 'openai' | 'claude' | 'deepl' | 'google-translate'
