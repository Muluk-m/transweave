    - Also has `generateText()` method for key generation

    **Step 6: Create DeepL provider** at `packages/server/src/ai/providers/deepl.provider.ts`:
    - Implements `TranslationProvider`
    - Constructor takes `apiKey: string`. Detects free vs pro key (free keys end with `:fx`) and sets base URL accordingly: `https://api-free.deepl.com/v2` for free, `https://api.deepl.com/v2` for pro
    - `translate()`: Makes one HTTP request per target language (DeepL does not support multi-target in one call). Uses `fetch()` with `Authorization: DeepL-Auth-Key {apiKey}` header. POST to `/translate` with `{ text: [text], source_lang, target_lang }`
    - Include `mapLanguageCode()` private method mapping ISO codes to DeepL format: `'en'->'EN'`, `'zh'->'ZH'`, `'zh-CN'->'ZH-HANS'`, `'zh-TW'->'ZH-HANT'`, `'pt'->'PT-PT'`, `'pt-BR'->'PT-BR'`, fallback to `code.toUpperCase()`
    - `validateApiKey()`: GET `/usage` with auth header, return `response.ok`
    - Does NOT support key generation (not an LLM). `generateText` should not exist on this provider.

    **Step 7: Create Google Translate provider** at `packages/server/src/ai/providers/google-translate.provider.ts`:
    - Implements `TranslationProvider`
    - Uses Google Cloud Translation API v2 basic (API key auth, no OAuth needed)
    - `translate()`: POST to `https://translation.googleapis.com/language/translate/v2` with `{ q: text, source: from, target: targetLang, key: apiKey, format: 'text' }`. One request per target language.
    - `validateApiKey()`: Make a test translation of "hello" from "en" to "es", return true if 200
    - Does NOT support key generation.

    **Step 8: Create provider factory** at `packages/server/src/ai/providers/provider-factory.ts`:

    - Export `isLLMProvider(provider: string): boolean` -- returns true for 'openai' and 'claude' (they support key generation), false for 'deepl' and 'google-translate'

    **Step 9: Create encryption utility** at `packages/server/src/ai/encryption.util.ts`:
    - Use `node:crypto` for AES-256-GCM encryption
    - `getEncryptionKey()`: reads `AI_ENCRYPTION_KEY` from env, derives 32-byte key via `scryptSync(secret, 'qlj-i18n-ai-salt', 32)`. Throws clear error if env var is missing.
    - `encryptApiKey(plaintext: string): string` -- returns `iv:tag:encrypted` (all base64 encoded)
    - `decryptApiKey(ciphertext: string): string` -- parses the format, decrypts. If decryption fails, throw a descriptive error "AI provider API key could not be decrypted. The encryption key may have changed. Please re-enter your API key in settings."
    - `maskApiKey(key: string): string` -- returns `'...{last4}'` for display in API responses. If key is shorter than 4 chars, return `'****'`.
