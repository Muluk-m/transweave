   **Quick Start with Docker (3 steps):**
   ```
   1. Clone the repository
      git clone https://github.com/your-org/qlj-i18n.git
      cd qlj-i18n

   2. Configure environment
      cp .env.example .env
      # Edit .env: set POSTGRES_PASSWORD and JWT_SECRET (required)
      # Generate JWT_SECRET: openssl rand -base64 64

   3. Start the platform
      docker compose up -d
      # Open http://localhost:3000 in your browser
      # On first launch, you'll see a setup wizard to create your admin account
   ```

   **Quick Start without Docker (PGlite - zero dependencies):**
   ```
   1. Clone and install
      git clone https://github.com/your-org/qlj-i18n.git
      cd qlj-i18n
      pnpm install

   2. Configure environment
      cp .env.example packages/server/.env
      # Edit .env: set JWT_SECRET (required)
      # Do NOT set DATABASE_URL -- PGlite will be used automatically
      # Generate JWT_SECRET: openssl rand -base64 64

   3. Start the backend
      pnpm dev:server

   4. Start the frontend (in a new terminal)
      pnpm dev:web
      # Open http://localhost:3000 in your browser
   ```
