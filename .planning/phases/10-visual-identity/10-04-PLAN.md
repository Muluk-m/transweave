**Step 2: Create and run generation script.**

Create `scripts/generate-og.mjs`:

```javascript
// scripts/generate-og.mjs
import sharp from 'sharp';
import { mkdirSync } from 'fs';

// Ensure assets dir exists
mkdirSync('assets', { recursive: true });

const svgSource = 'scripts/og-image-source.svg';

// Generate GitHub social preview (1280x640)
await sharp(svgSource)
  .resize(1280, 640)
  .png({ quality: 95 })
  .toFile('assets/og-image-github.png');
console.log('GitHub social preview generated: assets/og-image-github.png (1280x640)');

// Generate OG meta image (1200x630)
await sharp(svgSource)
  .resize(1200, 630, { fit: 'cover', position: 'center' })
  .png({ quality: 95 })
  .toFile('packages/web/public/og-image.png');
console.log('OG meta image generated: packages/web/public/og-image.png (1200x630)');

console.log('Done. Review the images before committing.');
```

Run: `node scripts/generate-og.mjs`

**Step 3: Review output images**, commit both PNG files, then remove scripts:
