# QLJ I18N Manager

## 项目简介

QLJ I18N Manager 是一个用于管理多语言资源的平台，帮助开发团队高效地管理多语言项目并实现无缝协作。

## 项目结构

```bash
.
├── packages/
│   ├── server/
│   └── web/
├── README.md
└── package.json
```


## 安装依赖

```bash
pnpm install
```

## 运行项目

```bash
cd packages/server
pnpm run start:dev
```

```bash
cd packages/web
pnpm run dev
```

```bash
docker compose up -d
```
