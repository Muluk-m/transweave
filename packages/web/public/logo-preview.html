Logo Concepts

4 个方向，都强调"多语言"概念。点击查看你喜欢哪个。

A

Globe

经典的国际化符号。经线、纬线、赤道——直观传达"全球多语言"。

B

Bubbles

两个重叠的语言块，一前一后。代表源语言与目标语言的对话和转换。

C

Layers

三层叠加的语言卡片，层层递进。同一份内容的多个语言版本，由浅到深。

D

Brackets

代码风格的尖括号 < > 包裹一个内容点。i18n key 的视觉隐喻，面向开发者。
